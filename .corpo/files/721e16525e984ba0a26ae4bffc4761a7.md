Checks the npm registry for the latest published version, compares it with
the locally installed version, and runs `npm install -g` if a newer version
is available.

## Invocation

```bash
corpo update
```

**Already up to date:**
```json
{ "current": "0.1.0", "latest": "0.1.0", "updated": false, "message": "Already up to date (0.1.0)." }
```

**Updated:**
```json
{ "current": "0.1.0", "latest": "0.2.0", "updated": true }
```

## Behavior

1. Fetches `https://registry.npmjs.org/@ilikesymmetry/corpo/latest`
2. Compares `version` field with the version embedded in `package.json` at build time
3. If equal: returns no-op result
4. If newer available: runs `npm install -g @ilikesymmetry/corpo@latest` via `Bun.spawnSync` with inherited stdio so npm output streams to the terminal

## Scope

Only works for users who installed via `npm install -g @ilikesymmetry/corpo`.
Contributors running from source should use `git pull` instead.

## Implementation

`src/commands/update.ts`
`package.json` — version read via static import at startup
