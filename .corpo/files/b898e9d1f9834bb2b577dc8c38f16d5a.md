# Session 2 — 2026-03-09

Third block of work. Thread resolution, documentation cleanup, and `corpo new`.

### Thread resolution

Human reviewers can now resolve threads directly from the GUI. Hover any
thread card to reveal a 14px SVG checkmark in the top-right corner. Clicking it:

1. Optimistically removes the thread from local state (no flicker)
2. Calls `DELETE /api/files/:fileId/threads/:threadId`
3. Server removes the `threads.{id}` frontmatter entry and `<!-- thread:{id} -->`
   body anchor, then auto-commits the file with message
   `thread resolved: {threadId} ({fileId})`

The checkmark is `absolute top-2 right-2` inside a `relative group/card`
container — zero layout shift whether visible or hidden.

Also added `corpo resolve <fileId> <threadId>` as a CLI command (plain file
write, no auto-commit — consistent with `corpo reply`). The API endpoint is
the only path that auto-commits, because resolution is a semantic milestone
worth a discrete commit.

### Thread anchor integrity lint + PostToolUse hook

Added `corpo lint [fileId]` to validate 1:1 correspondence between frontmatter
threads and body anchors. Two rules: `orphaned-thread` and `dangling-anchor`.
Code fences are stripped before scanning to avoid false positives from
documentation examples.

Registered as a Claude Code `PostToolUse` hook in `.claude/settings.json`:
fires automatically after every Edit or Write on a `.corpo/files/*.md` file.
Errors surface in the conversation immediately, before any git commit.

Also added `corpo threads <fileId>` — lists all threads as JSON with a
`needsResponse` flag (true when the last author isn't `claude`). Used by the
`resolve-threads` skill to find actionable threads.

### `corpo new` and shared createCorpoFile()

Added `corpo new <title> <description>` to scaffold a new corpo file without
reaching for Python. Outputs `{ id, path }`.

Both `corpo new` and `POST /api/files` now call `createCorpoFile(root, raw)`
from `src/lib/files.ts`. That function owns the ID generation + write logic;
neither callsite duplicates it.
