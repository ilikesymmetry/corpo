|---|---|---|
| File ID format | Resolved | UUID v4, hyphens stripped, 32 lowercase hex chars. Globally unique without coordination. |
| Thread anchor syntax | Resolved | HTML comment anchors (`<!-- thread:{id} -->`). Invisible to all Markdown renderers, unambiguous to parse. |

| Auth implementation (Okta) | Deferred | GitHub OAuth ships first; Okta follows |
| GUI / editor implementation | Deferred | Custom web app; must render threads natively |

## Out of Scope

- **Linear / GitHub replacement** — this system aggregates and links to them,
  it does not replace them
- **Real-time collaboration** — the loop is async: write, comment, delegate,
  revise
- **Custom permission system** — repo permissions are the primitive; no
  parallel ACL layer
- **Google Docs as an ongoing integration target** — Google Docs is an import
  source only; not a live sync target
