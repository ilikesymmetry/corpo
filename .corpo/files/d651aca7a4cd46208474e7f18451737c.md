---

The thread panel is a fixed-width aside on the right side of the file view.
It displays all threads in frontmatter order.

## Thread Cards

Each card shows:
- Author name and formatted timestamp
- Thread body text
- Replies (indented with a left border, smaller text)
- Reply input (when the thread is active)
- Resolve checkmark (on card hover, no layout shift)

## Activation

Threads activate from two places. Only the **opposite** side scrolls:

- **Click heading** → `activateFromHeading()` → `setPanelScrollTick(t => t+1)`
  → thread panel smoothly scrolls to center the card
- **Click thread card** → `activateFromThread()` → `setArticleScrollTick(t => t+1)`
  → article smoothly scrolls to center the heading

Two separate tick counters (`panelScrollTick`, `articleScrollTick`) ensure the
clicked side never scrolls. Each `useEffect` depends on only its own counter.

## Threaded Heading Highlights

Headings with attached threads render with a subtle amber background tint. Hover
intensifies it. Clicking activates the thread.

`parseHeadingThreads()` returns `Record<lineNumber, threadId[]>` keyed on source
line. `MarkdownRenderer` matches headings via `el.dataset.line` (stamped by
markdown-it's source map rule), avoiding false matches from typographer quote
conversions on apostrophes.

## Reply Input

A single-line textarea appears at the bottom of the active thread card. Enter
submits; Shift+Enter inserts a newline. On submit, the reply is appended to
`localMeta.threads[id].replies` and committed via `PUT /api/files/:id/commit`.

The input only shows when the thread is **active** (not on hover) to avoid
height jitter during mouse movement.

## Thread Resolution

A 14px SVG checkmark appears in the top-right corner of each card on hover.
It is `absolute top-2 right-2` inside a `relative group/card` container —
zero layout shift on show or hide.

On click (propagation stopped):
1. Thread is removed from local state optimistically
2. `DELETE /api/files/:fileId/threads/:threadId` is called
3. The server removes frontmatter entry + body anchor and creates a git commit

## Collapse

A hover-reveal button on the left edge collapses the panel to zero width (smooth
CSS transition). State is self-contained in `ThreadPanel` — not lifted to parent.
