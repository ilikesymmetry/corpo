# X launch thread

Write the X thread that accompanies the demo video. The thread explains the
problem and solution for people who want more than the 45-second demo.

## Structure

1. **Hook + video**: the demo. One punchy line as the caption.
2. **The problem**: what corporate docs look like for agents today. Concrete
   and relatable — copy-pasting from Google Docs into prompts.
3. **What Corpo is**: 2-3 sentences. Markdown, git, persistent IDs. The
   collaboration workflow stays the same.
4. **The protocol**: what makes it different from just "markdown files in a
   repo." Threads, frontmatter, progressive disclosure, multi-repo.
5. **How to try it**: `curl -fsSL https://corpo.sh/install | sh`. Link to
   the repo.
6. **What's next**: `corpo new` (agent-assisted doc creation). Open to
   contributors.
